# HJtip
a wheel of tooltip use sass (HJproject)

# Install
```
npm install
gulp build
```

# Usage
```HTML
<link rel="stylesheet" type="text/css" href="./dist/hjtip.css" />
```
### bottom-right

<div class="hintPrefix">
    <a href="#" aria-label="bottom-right" class="hint--bottom-right">bottom-right</a>   
</div>
